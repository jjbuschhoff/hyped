Apply Index Spans Data Processor Config.

Let (i, j) be an index span over the span sequence A=[(b_k, e_k)]_k. Then the data processor computes the following output span:

(b, e) = ((A_i)_0, (A_(j-1))_1)

That is it computes the span (b, e) represented by (i, j) in the domain of A.

Note that this is effectively the inverse operation to the hyped.data.processors.spans.idx_spans.CoveredIndexSpans data processor.
