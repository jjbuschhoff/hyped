class data.processors.templates.jinja2.Jinja2(config: Jinja2Config)

Bases: BaseDataProcessor[Jinja2Config]

Jinja2 Template Data Processor Config.

Creates a new feature by applying a jinja2 template to the given datapoints.

Access to the specific example values and their features is given through FeatureKey instances and jinja2 filters. Specifically see the following example on how to access the respective values:

valA = "The value of A is {{ FeatureKey('A') | index_example }}."
featA = "The feature of A is {{ FeatureKey('A') | index_features }}."
